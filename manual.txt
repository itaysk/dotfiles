- Mac settings
	- Trackpad
		- enable tap to click
		- disable look up (middleclick will bind to 3 finger click)
	- Keyboard
		- add Hebrew input
		- enable automatically switch to document language
		- set opt+space language switch shortcut (cmd+space is for alfred, ctrl+space is for intellisense)
	- Accessibility
		- enable zoom with gesture
- vim
	- install vim-plug (not on brew)
	- run `:PlugInstall` in vim
