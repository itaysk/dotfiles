# System Preference

- Trackpad
	- Point & Click
		- Tap to click: Enable 
- Keyboard
	- Key Repeat: Fast
	- Keyboard navigation: Enable
	- Delay Until Repeat: Short
	- Input Sources
		- Automatically switch to document's input source: Enable
		- Use smart quotes and dashes: Disable
	- Keyboard shortcuts...
		- Modifier keys
			- Caps Lock Key -> Escape
			- (On external keyboard): Option Key -> Command Key, Command Key -> Option Key
- Accessibility
	- Zoom
		- enable Use scroll gesture with modifier keys to zoom: ^ Control
- Sound
	- Play feedback when volume is changed: Enable
- Control Center
	- Sound: Always show in menu bar
	- Wi-Fi: Don't show in menu bar
	- Spotlight: Don't show in menu bar
- Desktop & Dock
	- Mission control
		- Automatically rearrange Spaces based on most recent use: Disable
		- When switching to an application, switch to a Space with open windows for the application: Enable
		- Group windows by application: Enable
	- Hot Corners...
		- Lower-Right: Disable
